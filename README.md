# Ai-Powered-Interview-Coach

An AI-powered web application that helps users prepare for interviews by
providing practice questions, mock interviews, and feedback using AI.

This project is built using FastAPI, Python, and SQLite.

## Features

- User registration and login system
- Practice interview questions
- AI-based interview feedback
- Simple and user-friendly interface
- Stores user data using SQLite database

## Technologies Used

- Python
- FastAPI
- HTML, CSS
- Jinja2 Templates
- SQLite
- Git & GitHub

## Installation and Setup

1. Clone the repository:

   git clone https://github.com/pavanvikas-star/Ai-Powered-Interview-Coach.git

2. Go to the project folder:

   cd AI_interview_coach

3. Install dependencies:

   pip install -r requirements.txt

4. Run the application:

   uvicorn main:app --reload

5. Open in browser:

   http://127.0.0.1:8000

## Usage

- Register as a new user
- Login to your account
- Start practicing interview questions
- Get feedback from AI
- Improve your interview skills
